Saturday 22th of November
QuakeWorld CD Project Update

Mr.Noname from the QuakeWorld CD Project has released a new version of the bootable linux Quakeworld CD. New features in version 0.1.5 include SSH support, ALSA (Advanced Linux Sound Architecture), and a new simplified setup and configuration.

Additional details on the QuakeWorld CD Project may be obtained in our original news post dated several months ago, and on the official QWCD forums right here at QuakeWorld.nu.

ezQuake gone SourceForge!

ezQuake has become more serious as we now see it hosted by sourceforge. What is ezQuake? I see it as copy-paste client taking all the great features from other projects and putting them together in a blender, based on latest FuhQuake release.
